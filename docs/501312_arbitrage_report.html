5. 限额-溢价 关联规律

规律一：收紧 → 溢价快速攀升

  • 上市以来每次限额大幅收紧后1-3天内，溢价率必升
  • 2025-06限购200元 → 溢价率升至5%+
  • 2026-04限购100元 → 溢价率跃升至8%+
  • 2026-06限购100元 → 预计溢价率将快速攀升

规律二：放宽 → 溢价回归理性

  • 每次限额放宽，溢价率1-2周内回落至2%以下
  • 2026-05放宽至5000元 → 溢价率快速回落
  • 放宽日 = 套利最佳入场时机

规律三：限额阈值效应

  • 限额 ≤ 200元 → 溢价率 3-10%
  • 限额 500-2000元 → 溢价率 1-3%
  • 限额 ≥ 5000元 → 溢价率 <1%（套利空间消失）
  • 501312是唯一幸存的有申购通道的科技QDII-LOF

规律四：净值涨 ≠ 限额松

  • 净值上涨吸引更多申购 → QDII额度加速消耗
  • 额度用尽时，即使净值上涨也得收紧
  • 净值下跌 → 赎回增加 → 额度释放 → 可能放宽
  • 2026-05从100→5000元的放宽恰逢净值上涨期
